# Psihointegritet — projekat uputstva

- IGNORIŠI vezani Marysoll design system (`_ds/`). Ne učitavaj njegov bundle ni CSS. Ovaj projekat koristi sopstvenu Psihointegritet paletu po zahtevu korisnika:
  - #2E3B2E primarna tamnozelena (forest)
  - #8A9D82 žalfija (sage)
  - #C6D5A8 svetlozeleni akcenat (meadow)
  - #D1A48C topli sekundarni akcenat (warm)
  - #3A2E28 primarni tekst (coffee)
  - #FAF8F3 glavno platno (canvas)
- Fontovi: Newsreader (naslovi, serif) + Instrument Sans (UI/body).
- Jezik: srpski (latinica), ijekavica u stručnim tekstovima terapeuta.
- Uzor za content rhythm: eyebrow → veliki editorial naslov → 2–3 rečenice → kartice → kome je namenjeno → CTA; dosta whitespace-a, sekcije kao kartice.
- Krajnji cilj: Next.js 16+ / React 19+ / Tailwind (semantic tokeni, bez inline CSS), struktura foldera definisana u prvoj poruci korisnika.
